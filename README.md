# ⏳ Sand Sim

Un simulateur de chute de sable interactif et performant, développé en **Rust**. Ce projet utilise les principes des automates cellulaires pour recréer la physique et le comportement granulaire des particules en temps réel.

## ✨ Fonctionnalités

* Simulation fluide et dynamique de particules (sable, etc.).
* Hautes performances grâce à la gestion mémoire de Rust.
* **[À compléter]** Interactions avec d'autres éléments (eau, murs, feu).

## 🛠️ Prérequis

Assurez-vous d'avoir installé [Rust et Cargo](https://www.rust-lang.org/tools/install) sur votre machine. Le projet a été testé avec succès sur la version **1.94+**.
